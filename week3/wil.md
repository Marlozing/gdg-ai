# 3주차 강의 내용  
결정 계수 R^2 : score 계산 시 사용  
MAE : 평균(|실제값-예측값|)  
과대적합 : train 데이터에 과도하게 적합해 test 데이터를 잘 평가하지 못하는 현상  
과소적합 : 훈련 수가 적어 train 데이터의 규칙을 찾지 못한 현상  
선형 회귀 : ax + b 형식의 직선을 학습하는 알고리즘  
특성 공학 : x^2과 같은 새로운 특성을 추가하는 방법  
ㄴ다항 회귀 : ax^2 + bx + c 형식의 곡선을 학습하는 알고리즘  
ㄴ다중 회귀 : ax1+ bx2 + c 형식의 여러 개의 특성을 학습하는 알고리즘  
Ridge 규제 : 모든 계수의 값을 낮게 해 과적합 방지  
Lasso 규제 : 특정 계수의 값을 0으로 해 과적합 방지  

# 느낀 점  
선형 회귀와 다항 회귀에 해대서는 대략적으로 알고 있었으나, AI의 기준으로는 다항 회귀 또한 여러 변수를 쓰는 선형 회귀와 같다는 것을 알게 되고 놀랐다.  
또한 과적합을 방지하는 방법은 단순히 훈련 횟수를 줄이는 방법만 알고 있었는데, Ridge나 Lasso와 같은 규제 방식을 통해 과적합을 방지할 수도 있다는 것을 알게 되었다.
